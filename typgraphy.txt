# Complete Font Size & Style Guide for User Experience

## Display & Hero Text
**Purpose**: Large impact text, hero sections, landing page headlines
- **Font Size**: 48-96px (3-6rem)
- **Font Weight**: 700-900 (Bold to Black)
- **Line Height**: 1.1-1.2
- **Letter Spacing**: -0.02em to -0.05em (tighter)
- **Recommended Fonts**: Inter, Poppins, Montserrat, Playfair Display
- **Use Cases**: Hero headlines, main page titles, marketing banners

## Primary Headings (H1)
**Purpose**: Main page titles, article headlines
- **Font Size**: 32-48px (2-3rem)
- **Font Weight**: 600-800 (Semibold to Extrabold)
- **Line Height**: 1.2-1.3
- **Letter Spacing**: -0.01em to -0.03em
- **Recommended Fonts**: Inter, Open Sans, Roboto, Source Sans Pro
- **Use Cases**: Page titles, blog post headlines, section headers

## Secondary Headings (H2)
**Purpose**: Major section divisions, category headers
- **Font Size**: 24-32px (1.5-2rem)
- **Font Weight**: 600-700 (Semibold to Bold)
- **Line Height**: 1.3-1.4
- **Letter Spacing**: -0.01em to -0.02em
- **Recommended Fonts**: Inter, Lato, Nunito Sans, Work Sans
- **Use Cases**: Section titles, product category headers, chapter titles

## Tertiary Headings (H3)
**Purpose**: Subsection headers, card titles

- **Font Weight**: 500-600 (Medium to Semibold)
- **Line Height**: 1.4-1.5

- **Recommended Fonts**: Inter, Roboto, Source Sans Pro, Fira Sans
- **Use Cases**: Card titles, subsection headers, feature titles

## Accessibility Guidelines
- **Minimum font size**: 16px for body text on mobile
- **Minimum contrast ratio**: 4.5:1 for normal text, 3:1 for large text
- **Large text definition**: 18px+ regular weight or 14px+ bold weight
- **Line height**: Never less than 1.2, ideally 1.5+ for body text
- **Font weight**: Avoid weights below 400 for body text

## Typography Pairing Recommendations
### Modern & Clean
- **Primary**: Inter (headings and UI)

- **Accent**: Roboto Mono (code/numbers)

### Professional & Readable
- **Primary**: Work Sans (headings)
- **Secondary**: Open Sans (body text)
- **Accent**: Source Code Pro (technical text)

### Elegant & Contemporary
- **Primary**: Poppins (headings)
- **Secondary**: Nunito Sans (body text)
- **Accent**: Fira Code (code)

### Classic & Timeless
- **Primary**: Source Sans Pro (headings)
- **Secondary**: Georgia (body text)
- **Accent**: Courier New (monospace)

## Responsive Typography Scale
### Desktop (1200px+)
- Scale ratio: 1.25 (Major Third)
- Base size: 16px

### Tablet (768px - 1199px)
- Scale ratio: 1.2 (Minor Third)
- Base size: 16px

### Mobile (320px - 767px)
- Scale ratio: 1.15
- Base size: 16px (minimum)

## Performance Considerations
- **Font loading**: Use font-display: swap for better performance
- **Font subsets**: Load only necessary character sets
- **Variable fonts**: Consider for fewer HTTP requests
- **System fonts**: Fallback to system UI fonts for instant rendering

## Color & Typography Interaction
- **Light backgrounds**: Use darker text (minimum #333333)
- **Dark backgrounds**: Use lighter text (minimum #CCCCCC)
- **Colored backgrounds**: Ensure sufficient contrast
- **Link text**: Differentiate with color and/or underline
- **Interactive states**: Clear hover/focus indicators

This comprehensive guide covers all major UI text elements with specific 
recommendations for font sizes, weights, and spacing to ensure optimal user experience across all devices and use cases.
